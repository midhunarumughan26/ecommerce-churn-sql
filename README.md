# E-Commerce Customer Churn Analysis (SQL)

## 📌 Overview
This project analyzes 10,000+ customer records to identify churn patterns using SQL.

## 🔧 Tools Used
- SQL (MySQL / PostgreSQL)
- Advanced SQL techniques:
  - JOINs
  - Subqueries
  - CASE statements
  - GROUP BY

## 📊 Key Analysis
- Churn distribution across tenure
- Impact of satisfaction score on churn
- Payment mode influence on churn
- Complaint-driven churn patterns
- Return behavior of churned customers

## 🚀 Key Insights
- Customers with complaints are significantly more likely to churn
- Low satisfaction scores strongly correlate with churn
- Certain payment modes show higher churn rates
- High return frequency is linked to churned customers

## 📈 Business Recommendations
- Improve complaint resolution speed
- Target low satisfaction customers with retention offers
- Optimize payment experience
- Monitor return-heavy customers proactively

## 📂 Project Structure
- schema.sql → Table creation
- data_cleaning.sql → Data preprocessing
- churn_analysis.sql → Core churn insights
- return_analysis.sql → Behavioral insights
- insights.sql → Business-focused queries
